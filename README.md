# WASM Error Propagation Dither Comparison

A comparison of error propagation dither performance between JavaScript and WebAssembly generated from C++, D, Rust and Zig.

## Dependencies

* node >= 16.14
* npm

## Getting Started

* `npm install`
* `npm start`
